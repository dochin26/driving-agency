# 要件定義レビュー

CLAUDE.mdの要件定義をレビューして、以下の観点から問題点や改善提案を行ってください：

## レビュー観点

1. **機能要件の完全性**
   - 必要な機能が全て定義されているか
   - 矛盾や抜け漏れがないか

2. **データベース設計の妥当性**
   - テーブル構造は正規化されているか
   - インデックスは適切か
   - 外部キー制約は妥当か

3. **画面設計の使いやすさ**
   - ユーザー導線は適切か
   - モバイルでの操作性は考慮されているか

4. **技術選定の妥当性**
   - 選定した技術スタックは要件に適しているか
   - 学習コストとメリットのバランスは取れているか

5. **セキュリティ**
   - 認証・認可は適切か
   - データ保護は十分か

## 出力形式

- 問題点や改善提案を箇条書きで提示
- 優先度（高・中・低）を明記
- 具体的な改善案を提示

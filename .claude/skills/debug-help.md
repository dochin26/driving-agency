# デバッグ支援

エラーやバグの原因を特定し、解決策を提示してください。

## デバッグ手順

1. **エラーメッセージの分析**
   - エラーの種類を特定
   - スタックトレースから原因箇所を特定

2. **原因の推測**
   - よくある原因パターンと照合
   - 関連するコードを確認

3. **解決策の提示**
   - 複数の解決策を提示（優先度順）
   - それぞれのメリット・デメリット

4. **再発防止策**
   - テストの追加
   - バリデーションの追加
   - コードの改善

## 対象エラー例

### Rails固有
- ルーティングエラー
- ActiveRecord エラー（RecordNotFound, RecordInvalid）
- テンプレートエラー
- アソシエーションエラー

### データベース
- マイグレーションエラー
- 制約違反
- N+1クエリ問題

### JavaScript（Stimulus）
- イベントリスナー未動作
- データ属性の誤り

### 環境関連
- Gem依存関係
- Docker環境の問題
